# Azure Virtual Machine Deployment

## Objective
- Create a basic Azure Virtual Machine.
- Access the VM using RDP (Windows) or SSH (Linux).
- Understand Azure compute fundamentals.

## Steps
1. Log in to the Azure Portal and create a **Virtual Machine**.
2. Choose OS image (Windows/Linux), VM size, and authentication method.
3. Configure networking and security rules.
4. Launch the VM and access it via **RDP** or **SSH**.
5. Verify successful login and VM operation.
