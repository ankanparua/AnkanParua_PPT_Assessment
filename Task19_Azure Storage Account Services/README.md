# Azure Storage Account Services

## Objective
- Create an Azure Storage Account.
- Explore Blob, File, and Table storage.
- Upload and manage sample data in each service.

## Steps
1. Log in to the Azure Portal and create a **Storage Account**.
2. Create a **Blob container** and upload sample files.
3. Create a **File Share** and add sample data.
4. Create a **Table** and insert sample entities.
5. Verify data access and storage configuration.
