# Azure Virtual Network Setup


## Objective
- Create an Azure Virtual Network.
- Configure subnets within the VNet.
- Apply NSG rules for traffic control.

## Steps
1. Log in to the Azure Portal and create a **Virtual Network (VNet)**.
2. Define the address space and create one or more **subnets**.
3. Create a **Network Security Group (NSG)**.
4. Add inbound and outbound security rules (e.g., allow SSH/RDP).
5. Associate the NSG with the subnet or network interface.
6. Verify network configuration and access.
