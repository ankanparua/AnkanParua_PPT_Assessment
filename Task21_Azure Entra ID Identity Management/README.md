# Azure Entra ID Identity Management


## Objective
- Create and manage users in Azure Entra ID.
- Assign built-in roles for access control.
- Understand identity and role-based access management.

## Steps
1. Log in to the Azure Portal and navigate to **Azure Entra ID**.
2. Create new **users** with basic profile information.
3. Assign **roles** (e.g., Global Reader, User Administrator) to users.
4. Verify role assignments and access permissions.
5. Test login with created users (optional).
