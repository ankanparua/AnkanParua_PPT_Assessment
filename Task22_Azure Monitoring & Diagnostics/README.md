# Azure Monitoring & Diagnostics

## Objective
- Enable monitoring for Azure resources.
- View metrics and diagnostic data.
- Understand basic performance tracking in Azure.

## Steps
1. Log in to the Azure Portal and select an Azure resource (VM, VNet, or Storage).
2. Navigate to **Azure Monitor** from the portal.
3. Enable **Metrics** and **Diagnostics settings** for the resource.
4. View real-time and historical performance metrics.
5. Analyze basic health and usage data.
