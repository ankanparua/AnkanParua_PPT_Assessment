# Azure Resource Management & Cost Tracking

## Objective
- Create and manage Azure Resource Groups.
- Understand Azure Resource Manager (ARM) concepts.
- Monitor and analyze resource costs.

## Steps
1. Log in to the Azure Portal and create a **Resource Group**.
2. Deploy or move resources into the Resource Group.
3. Explore **Azure Resource Manager (ARM)** for centralized management.
4. Navigate to **Cost Management + Billing**.
5. View cost analysis and usage breakdown.
