# Azure SLA & Subscription Governance

## Objective
- Review Azure Service Level Agreements (SLA).
- Understand service lifecycle stages.
- Explore Azure subscription management.

## Steps
1. Log in to the Azure Portal and navigate to **Service Health** and **SLA documentation**.
2. Review SLA details for commonly used Azure services.
3. Explore **Subscription** settings and limits.
4. Understand service lifecycle stages (Preview, GA, Deprecated).
5. Verify subscription usage, roles, and policies.
