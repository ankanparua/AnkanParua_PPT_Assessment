# AWS IAM User & Group Management

## Project Overview
This project demonstrates creating and managing IAM users and groups in AWS to control access and enforce security policies.

## Objective
- Create IAM users for individuals or applications.
- Create IAM groups for role-based access.
- Attach policies to users or groups to define permissions.

## Steps
1. Log in to the AWS Management Console and navigate to **IAM**.
2. Create IAM users with appropriate access type (Programmatic, Console, or both).
3. Create IAM groups (e.g., Admins, Developers) and attach policies.
4. Add users to groups to inherit permissions.
5. Verify user access by logging in or using AWS CLI commands.
